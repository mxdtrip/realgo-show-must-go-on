# Web

Веб-приложение realgo на Next.js, React и TypeScript.

Текущая реализация состоит из маркетингового лендинга и гибридного личного кабинета.
Оформление пока построено на общем `app/globals.css` без подключения Tailwind/shadcn/ui, чтобы
сохранить единую GitHub Primer dark эстетику.

## Маршруты

- `/` — маркетинговый лендинг.
- `/dashboard` — обзор подготовки.
- `/reviews` — очередь повторений.
- `/problems` — личная база задач.
- `/roadmap` — план подготовки.
- `/patterns` — слабые паттерны.
- `/cards` — карточки повторения.
- `/cards/session` — полноэкранная сессия повторения без элементов кабинета.
- `/extension` — статус расширения.
- `/settings` — настройки аккаунта и приватности.

Личный кабинет уже подключён к backend API для auth, `/reviews` и части настроек. Остальные
разделы всё ещё используют моки из `app/(cabinet)/_mock.ts` до отдельного wiring.

## PWA и уведомления

- `app/manifest.ts` описывает PWA-приложение.
- `public/sw.js` регистрируется через `app/_pwa/PWAProvider.tsx`.
- `/settings` содержит панель установки приложения и настройки уведомлений.
- Базовые флаги уведомлений синхронизируются с backend, browser permission/reminder time и
  тестовое уведомление остаются локальными через service worker. Push-уведомления подключаются
  отдельной задачей.

## Локализация

Тексты и моковые записи вынесены в `app/_content/i18n.ts`. Сейчас заведены языки `ru`, `en`
и `es`; переводы для `en/es` пока не добавлены и используют те же source-строки, чтобы позже
заменить значения без переписывания компонентов.
