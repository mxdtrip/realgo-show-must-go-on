# API

Go backend-сервис.

```text
cmd/api/       точка входа и сборка зависимостей
internal/      бизнес-логика, transport, storage и интеграции
migrations/    версионируемые миграции базы данных
```

`go.mod` следует создать в `services/api`, когда будет определён canonical module path. Если появятся другие Go-сервисы, каждый получает собственный `go.mod`, а корневой `go.work` объединяет их для локальной разработки.
